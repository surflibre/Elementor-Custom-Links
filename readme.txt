Description:
This plugin adds a custom link field to images in the WordPress media library and automatically applies these links to all images in Elementor Image Carousel widgets. Lightbox is disabled for these images automatically.

Installation:
1. Upload the plugin folder to /wp-content/plugins/ or install via WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit an image in the Media Library and add a custom link using the new 'Custom Link' field.
4. The link will automatically replace the default href in all Elementor Image Carousels.

Frequently Asked Questions:
Q: Does this work with multiple carousels on the same page?
A: Yes, the plugin applies the custom link to each slide individually.

Q: Can I open links in a new tab?
A: Yes, there is a checkbox in the media library to open the link in a new tab.
